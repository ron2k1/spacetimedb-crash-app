# 3D model credits

## Fox.glb

`Fox.glb` is the Khronos **Fox** sample asset. A fox is Crash's mascot -- kept as the app icon
and the in-app mascot mark in homage to the project's original fox guide. The large interactive
guide on the dashboard stage is a Spline robot (loaded at runtime), so this `Fox.glb` mesh is not
the on-stage 3D model; it is retained in the model folder and consumed by the Unity parity client
(glTFast), and it still travels into distributed bundles, so its attribution is preserved here.

- **Mesh:** by **PixelMannen** -- released into the public domain (CC0 1.0).
- **Rig + animations** (Survey / Walk / Run): by **tomkranis** -- licensed **CC-BY 4.0**
  (<https://creativecommons.org/licenses/by/4.0/>).
- **Source:** Khronos glTF Sample Assets --
  <https://github.com/KhronosGroup/glTF-Sample-Assets/tree/main/Models/Fox>

The CC-BY 4.0 license on the rig/animation requires attribution. Crash satisfies this with
this CREDITS file, which travels with the model in the repo and in any built `dist/` that
bundles the asset.

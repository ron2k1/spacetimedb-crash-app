# Web Search

This plugin gives the assistant the ability to search the web. Without it, the assistant can only rely on what it already learned during training, which means it may not know about recent events, new prices, or anything that changed after its training. With this plugin enabled, it can look things up and answer using current information, and it can tell you where the answer came from.

A beginner would turn this on when they want answers about things happening now, like today's news, a current product, or a fact that tends to change over time. If you find the assistant saying it is not sure or that its information might be out of date, enabling Web Search is usually the fix.

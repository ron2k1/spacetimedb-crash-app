# PDF Reader

This plugin lets the assistant open a PDF and read the text inside it. Once it is enabled, you can point the assistant at a document and ask for a plain-language summary, a list of the main points, or the answer to a specific question buried in the file.

A beginner would enable this when they have a long PDF they do not want to read end to end, like a syllabus, a contract, a manual, or a research paper. Instead of scrolling through every page, you let the assistant read it and ask it what you actually need to know.

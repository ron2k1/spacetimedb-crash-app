# What is a skill?

A skill is a small, reusable instruction set that teaches your AI assistant how to do one job well. Think of it like a saved recipe. Instead of explaining from scratch every time how you want your meeting notes cleaned up, you save those instructions once as a skill and reuse it whenever you need it.

Each skill lives in its own folder and has a short description that says when to use it. When the job matches that description, the assistant follows the steps in the skill. Skills keep your assistant consistent, so you get the same quality result every time rather than something slightly different on each try.

A plugin is a little different. A skill teaches the assistant how to do something, while a plugin gives it a new ability it did not have before, like searching the web or reading a PDF. You will see both in Crash. Skills are the recipes, and plugins are the new tools in the kitchen.

In the next steps you will install a ready-made skill from the marketplace and then build one of your own. You do not need to write any code to do either one.

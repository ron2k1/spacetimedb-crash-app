# Install from the marketplace

The fastest way to get started is to install something that already exists. Crash has two places to browse. The Skills Marketplace holds ready-made skills like Meeting Notes and Study Buddy. The Plugin Marketplace holds plugins like Web Search and PDF Reader that give your assistant new abilities.

Open one of those dashboards and look through the list. Each item shows its name and a short description telling you when to use it. When you find one you want, click install. There is nothing to download from the internet and nothing to set up. The item is already on your machine, ready to go.

When you install something, Crash copies that item's folder into your working folder. This is worth understanding because it is not magic. After you install, you can open your working folder and watch the new files appear right there. That folder is yours, the files are plain text, and you can open them and read exactly what the skill or plugin contains.

Try installing one skill and one plugin now, then peek at your working folder to see the new files show up. Once you are comfortable browsing and installing, move on to the last step where you make a skill of your own.
